Добро пожаловать в морской бой 1.0
Перед началом игры задайте параметры боя - набор кораблей, размер поля, вертикальное размещение
При игре в морской бой ваш соперник не должен видеть ваших кораблей. Для этого сделано "переходное окно"
Правила как в обычном морском бое, исключение - вы можете задать размещение кораблей по диагонали
